# Abscond Mobile (iOS) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a separate `abscond-mobile` GitHub repo that wraps the existing Vue PWA in a Capacitor WKWebView shell, adds native iOS background audio + lock screen + haptics, and produces a signed-off unsigned IPA via GitHub Actions on every `v*` tag.

**Architecture:** `abscond-mobile` holds only Capacitor scaffolding, a Swift native plugin, and CI config — the Vue app lives in `abs-ui` as a git submodule. On tag push, the workflow checks out submodules, builds the Vue app, syncs it into the iOS project, builds an unsigned `.xcarchive`, packages it as an IPA, and attaches it to a GitHub Release. Feather (or any re-signing tool) then signs the IPA for sideloading.

**Tech Stack:** Capacitor 7, Swift 5, AVFoundation, UIKit (`UIImpactFeedbackGenerator`), CocoaPods, GitHub Actions macOS-15 runner (`xcodebuild`), `gh` CLI for release creation.

```swift
    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {

        return true

    ...
}
```

- [ ] **Step 2: Add AVFoundation import and configure audio session**

Replace the file's top imports and the `didFinishLaunchingWithOptions` method with:

```swift
import UIKit
import Capacitor
import AVFoundation

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

    var window: UIWindow?

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        do {
            try AVAudioSession.sharedInstance().setCategory(
                .playback,
                mode: .spokenAudio,
                options: [.allowBluetooth, .allowBluetoothA2DP]
            )
            try AVAudioSession.sharedInstance().setActive(true)
        } catch {
            print("AVAudioSession setup failed: \(error)")
        }
        return true
    }
```

Leave all other generated delegate methods (`applicationWillResignActive`, etc.) unchanged.

## Task 5: HapticsBridgePlugin

**Files:**
- Create: `ios/App/App/HapticsBridgePlugin.swift`
- Create: `ios/App/App/HapticsBridgePlugin.m`

> The existing Vue app calls `navigator.vibrate(30)` in ~20 places. On iOS PWA and WKWebView, `navigator.vibrate` is a no-op. This plugin: (1) overrides `navigator.vibrate` via a `WKUserScript` injected at document start so existing Vue code works unchanged, and (2) exposes a `HapticsBridge.impact()` method that routes to `UIImpactFeedbackGenerator` for true native haptics.

- [ ] **Step 1: Create HapticsBridgePlugin.swift**

```swift
// ios/App/App/HapticsBridgePlugin.swift
import Capacitor
import UIKit
import WebKit

@objc(HapticsBridgePlugin)
public class HapticsBridgePlugin: CAPPlugin {

    // Called once when the Capacitor bridge initialises — before the webview loads any page.
    // We inject a WKUserScript at document-start that replaces the no-op navigator.vibrate
    // with a call into this plugin so existing Vue code works without modification.
    override public func load() {
        guard let contentController = bridge?.webView?.configuration.userContentController else {
            return
        }
        let script = WKUserScript(
            source: """
            navigator.vibrate = function() {
                window.Capacitor?.Plugins?.HapticsBridge?.impact({ style: 'light' });
            };
            """,
            injectionTime: .atDocumentStart,
            forMainFrameOnly: true
        )
        contentController.addUserScript(script)
    }

    @objc func impact(_ call: CAPPluginCall) {
        let style = call.getString("style") ?? "light"
        DispatchQueue.main.async {
            let feedbackStyle: UIImpactFeedbackGenerator.FeedbackStyle = {
                switch style {
                case "heavy":  return .heavy
                case "medium": return .medium
                default:       return .light
                }
            }()
            let generator = UIImpactFeedbackGenerator(style: feedbackStyle)
            generator.prepare()
            generator.impactOccurred()
        }
        call.resolve()
    }

    @objc func notification(_ call: CAPPluginCall) {
        let type = call.getString("type") ?? "success"
        DispatchQueue.main.async {
            let feedbackType: UINotificationFeedbackGenerator.FeedbackType = {
                switch type {
                case "warning": return .warning
                case "error":   return .error
                default:        return .success
                }
            }()
            let generator = UINotificationFeedbackGenerator()
            generator.notificationOccurred(feedbackType)
        }
        call.resolve()
    }
}
```

- [ ] **Step 2: Create HapticsBridgePlugin.m (Capacitor plugin registration)**

```objc
    CAP_PLUGIN_METHOD(notification, CAPPluginReturnPromise);
)
```

## Task 6: GitHub Actions Workflow

**Files:**
- Create: `.github/workflows/build-ipa.yml`
- Create: `ExportOptions.plist`

> On every `v*` tag push, the workflow: checks out with submodules, builds the Vue app, syncs into iOS, installs CocoaPods, builds an unsigned `.xcarchive`, packages it into an IPA manually (no export signing), and creates a GitHub Release with the IPA attached.

- [ ] **Step 1: Create ExportOptions.plist**

This file is read by `xcodebuild -exportArchive`. For an unsigned build we skip it entirely — instead we manually zip the `.app` from the archive. Create a placeholder so the path exists:

```bash
mkdir -p .github/workflows
```

- [ ] **Step 2: Create .github/workflows/build-ipa.yml**

```yaml
# .github/workflows/build-ipa.yml
name: Build IPA

on:
  push:
    tags:
      - 'v*'

jobs:
  build:
    runs-on: macos-15
    timeout-minutes: 60

    steps:
      - name: Checkout (with submodules)
        uses: actions/checkout@v4
        with:
          submodules: recursive

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '22'
          cache: 'npm'
          cache-dependency-path: abs-ui/package-lock.json

      - name: Install Capacitor CLI deps
        run: npm ci

      - name: Build Vue app
        run: cd abs-ui && npm ci && npm run build

      - name: Sync web assets into iOS project
        run: npx cap sync ios

      - name: Install CocoaPods dependencies
        run: cd ios/App && pod install

      - name: Build archive (unsigned)
        run: |
          set -o pipefail
          xcodebuild archive \
            -workspace ios/App/App.xcworkspace \
            -scheme App \
            -configuration Release \
            -destination "generic/platform=iOS" \
            -archivePath "$RUNNER_TEMP/Abscond.xcarchive" \
            CODE_SIGNING_REQUIRED=NO \
            CODE_SIGN_IDENTITY="" \
            CODE_SIGNING_ALLOWED=YES \
            DEVELOPMENT_TEAM="" \
            | xcpretty

      - name: Package IPA
        run: |
          TAG="${GITHUB_REF_NAME}"
          APP=$(find "$RUNNER_TEMP/Abscond.xcarchive/Products/Applications" -name "*.app" | head -1)
          mkdir -p "$RUNNER_TEMP/Payload"
          cp -r "$APP" "$RUNNER_TEMP/Payload/"
          cd "$RUNNER_TEMP"
          zip -qr "Abscond-${TAG}.ipa" Payload/
          echo "IPA_PATH=$RUNNER_TEMP/Abscond-${TAG}.ipa" >> "$GITHUB_ENV"
          echo "TAG=${TAG}" >> "$GITHUB_ENV"

      - name: Create GitHub Release and attach IPA
        env:
          GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          gh release create "$TAG" \
            "$IPA_PATH" \
            --title "Abscond $TAG" \
            --generate-notes \
            --repo "$GITHUB_REPOSITORY"
```
